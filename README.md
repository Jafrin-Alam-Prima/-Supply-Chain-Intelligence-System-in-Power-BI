# 📦 Supply Chain Intelligence Dashboard (Power BI)

## 📌 Project Overview

This project is a fully interactive Supply Chain Intelligence System built using Power BI to monitor end-to-end supply chain performance.

The dashboard provides actionable insights across inventory management, order fulfillment, logistics performance, warehouse operations, and vendor effectiveness.

The objective is to transform raw operational data into decision-support analytics that help reduce risk, improve delivery efficiency, and optimize inventory levels.

---

## 🎯 Business Problem

Organizations often struggle with:

* Late deliveries impacting customer satisfaction
* Stock-outs and excess inventory
* Poor vendor performance visibility
* Inefficient warehouse utilization
* Lack of real-time KPI monitoring

This dashboard addresses these challenges through structured data modeling and advanced DAX analytics.

---

## 🏗️ Data Model

* Star Schema Architecture
* Fact Table: Orders / Shipments
* Dimension Tables:

  * Date
  * Product
  * Vendor
  * Warehouse
  * Region

Best practices implemented:

* Role-playing Date table
* Optimized relationships
* Clean Power Query transformations

---

## 📊 Key KPIs & Metrics

* Total Orders
* Total Revenue
* On-Time Delivery %
* Inventory Turnover Ratio
* Order Cycle Time
* Lead Time vs Delivery Time
* Vendor Ranking
* Stock-Out Risk Indicator
* Supply Chain Risk Score (Custom Model)
* Scenario-Based Optimization (What-If Analysis)

---

## 📈 Dashboard Pages

1. Supply Chain Overview
2. Inventory & Stock Management
3. Order Fulfillment & Demand Trends
4. Logistics & Delivery Performance
5. Vendor & Supplier Analysis
6. Warehouse & Regional Insights
   

## 🛠️ Tools Used

* Power BI
